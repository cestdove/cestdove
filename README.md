## 🎓 Academic Research

> **"Classificazione automatica di ticket tramite rappresentazione TF-IDF in spazi vettoriali ad alta dimensionalità: un approccio Data-Centric con SVM"**

- **Research Focus** — NLP pipeline for automated helpdesk triage.
- **Data-Centric AI** — Implemented controlled label noise and semantic perturbations to improve model robustness.
- **Theory** — Deep dive into SVM soft-margin optimization, primal-dual formulation, and high-dimensional vector spaces.
- **Resources** — Documentation and implementation available in [PW_Triage_ML](https://github.com/cestdove/PW_Triage_ML).
