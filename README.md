## 🚀 Research & Projects

*   🎓 **[Data-Centric SVM Triage](https://github.com/cestdove/PW_Triage_ML)**: My university thesis on automated text classification. Proved that semantic data perturbation and noise injection yield higher performance gains than hyperparameter tuning. Built using TF-IDF and SVMs in high-dimensional spaces.
*   🪶 **[BioAcoustic Vector Search](https://github.com/cestdove/BioAcoustic-Vector-Search)**: End-to-end search engine for bird species identification utilizing MFCC extraction, ChromaDB, and a Streamlit UI.
*   👁️ **Current Focus**: Expanding modular ingestion and search scripts into **Computer Vision**, bridging neural architectures with traditional machine learning for complex visual analysis.
